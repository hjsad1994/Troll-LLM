## ADDED Requirements

### Requirement: Checkout Discord Integration Text
The checkout page Discord section SHALL display user-friendly messaging that highlights the benefits of joining Discord.

#### Scenario: English user views checkout page
- **WHEN** user views the checkout page in English
- **THEN** the Discord section shows "Join to get the fastest support, giveaways, and exclusive offers"

#### Scenario: Vietnamese user views checkout page
- **WHEN** user views the checkout page in Vietnamese
- **THEN** the Discord section shows "Tham gia để được hỗ trợ nhanh nhất, giveaway và các ưu đãi đặc biệt"
