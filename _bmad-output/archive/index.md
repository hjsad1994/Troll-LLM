# TrollLLM - Project Documentation Index

## Overview

TrollLLM is a high-performance LLM API proxy service providing unified access to multiple AI providers with advanced billing, rate limiting, and key management features.

**Project Type:** Multi-part (Frontend + Backend + GoProxy)
**Primary Languages:** TypeScript, Go
**Database:** MongoDB

## Technology Stack

### Frontend
- **Framework:** Next.js 14 (App Router)
- **UI:** React 18, TailwindCSS
- **Language:** TypeScript
- **i18n:** next-intl (en, vi, zh)

### Backend
- **Framework:** Express.js
- **Language:** TypeScript
- **Database:** MongoDB (Mongoose)
- **Auth:** JWT, bcryptjs

### GoProxy
- **Language:** Go 1.25
- **HTTP:** net/http with HTTP/2
- **Database:** MongoDB Driver
- **Features:** SSE streaming, proxy pooling

## Key Features

1. **Multi-Provider Support**
   - OpenAI-compatible API (`/v1/chat/completions`)
   - Anthropic-native API (`/v1/messages`)
   - Model-based upstream routing

2. **Billing System**
   - Token-based billing with cache pricing
   - Per-model billing multipliers
   - Credit expiration management

3. **Access Control**
   - User API keys (`sk-troll-*`)
   - Friend keys with model limits (`fk-*`)
   - Tier-based rate limiting (Dev/Pro)

4. **High Availability**
   - Key pool rotation with health checking
   - Proxy pool for IP rotation
   - Auto-reload configuration
